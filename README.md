# CocktailBotHAL
A Harware Abstraction Layer for cocktail mixing robots

WIP

## API
* Bot 
  * initialize
  * disable
  * clean
  * calibrate
  * setup liquids
  * configure e. g. logging..
  * mix drink
* Glass 
  * take
  * offer
  * test presence
* Mixer 
  * get number of liquids
  * get liquids names
  * select liquid
* Dispenser 
  * dispense n units
  * out of liqid
